# Broz Agency (broz.agency)

## Project Overview
Персональный сайт-визитка Броза (Евгения Киршина), Full Cycle Product Engineer и основателя сообщества «FlutterFlow Rus». Цель сайта — позиционирование как эксперта, лидера мнений и надежного партнера для бизнеса (MVP за 24 часа, полноценные продукты под ключ).

### Core Stack
- **Framework:** Flutter (FlutterFlow compatible)
- **Backend:** Supabase (Postgres with RLS)
- **Deployment:** Vercel
- **Automation:** n8n

### Design Philosophy (Linear Style)
- **Font:** Inter
- **Background:** #050505
- **Accent:** #6d5dfc
- **Rounding:** 28px (blocks) / 14px (elements)
- **Aesthetic:** Современный, «живой», полированный интерфейс с интерактивной обратной связью.

## Project Structure (Planned)
1. **Hero-блок:** «Перевожу задачи бизнеса на язык работающих IT-решений».
2. **Блок «Человек-команда»:** Описание этапов (Проектирование → Дизайн → Разработка → Автоматизация).
3. **Специализация:** Финтех (CoinLover), Недвижимость, Релокация (Baonline), SMB (Reloto).
4. **Экспертность:** Основатель «FlutterFlow Rus», отзывы, менторство.
5. **Портфолио:** Reloto, CoinLover, Baonline.
6. **Pricing:** MVP от $350, Продукт от $3,500.
7. **CTA:** Ссылка на Telegram.

## Core Mandates & Instructions
- **Clean Architecture:** Всегда использовать чистую архитектуру для Flutter action'ов.
- **SQL Safety:** Всегда использовать Row Level Security (RLS) для таблиц Supabase.
- **Currency:** Все цены указывать в USD.
- **Automation:** Использовать флаг `--output-format json` для n8n ворклоу.
- **Language:** Писать документацию и отвечать на русском языке.
